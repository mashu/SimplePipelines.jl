SimplePipelines.jl

Minimal, type-stable DAG pipelines for Julia

SimplePipelines is for small-to-medium workflows where the shape of the work should be visible in Julia code. A step can be a shell command or a Julia function. Operators connect steps into a DAG. run executes that DAG and returns one result per step.

The central idea is deliberately small:

step_a >> step_b        # run in order
step_a & step_b         # run branches in parallel
run(pipeline)           # execute and collect StepResult values

For a complete first example, start with Getting started.
